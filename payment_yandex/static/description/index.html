Yandex Checkout

The Yandex.Checkout Module is the ultimate payment solution for Odoo:
  • includes 12 payment acceptance methods
  • suitable for companies and entrepreneurs
  • settlements are made to the company`s bank account

How to connect

  1. Send a request and follow the instructions in Merchant Profile.
  2. Select the Payment module connection method. Read the module configuration manual.
  3. Set up the Yandex.Checkout Odoo Module.
  4. Make a test run to check if the payment notifications are sent correctly.

Onboarding with Yandex.Checkout and module configuration are free of charge.

View all rates at the Yandex Checkout website
